Weather Check App

Real-time weather updates using OpenWeather API

Project Overview

This Weather Check App allows users to enter any city and instantly get the current weather conditions, including temperature, humidity, and wind speed. It's powered by the OpenWeatherMap API and built using basic web technologies.

Features

  • Search weather by city name
  • Displays temperature, humidity, and wind speed
  • Live data fetched via OpenWeather API
  • Minimal, responsive interface

Technologies Used

  • HTML, CSS, JavaScript
  • API: OpenWeatherMap
  • Fetch API for data handling
