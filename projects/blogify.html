Blogify

A simple web-based blogging platform

Project Overview

Blogify is a user-friendly blogging platform that allows users to write, edit, and share blog posts easily. It mimics the functionality of popular blog CMS tools with a focus on simplicity and responsive design.

Features

  • Create and publish blog posts with title and content
  • Display all blogs in a user-friendly format
  • Database will be used for storing blogs
  • Responsive design for desktop and mobile views

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express
  • Database: MongoDB
