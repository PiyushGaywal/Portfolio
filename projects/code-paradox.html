Code-Paradox

An interactive live code collaboration platform

Project Overview

Code-Paradox allows users to write, run, and collaborate on code in real-time. Users can share coding sessions with unique room IDs and experience a live collaborative coding environment. It supports multiple programming languages and includes features like syntax highlighting and auto-bracket completion.

Features

  • Live code collaboration with multiple users
  • Run code in real-time in the browser
  • Syntax highlighting and auto-bracket completion
  • Share unique room links for sessions
  • Responsive design for all devices

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express
  • Tools: Web-socket, CodeMirror
