Hotel Booking App

A responsive and user-friendly hotel booking system

Project Overview

This Hotel Booking App is designed to allow users to register, explore hotel options, and make reservations. Admin features include hotel listing and booking management. Built using modern web technologies with MongoDB integration.

Features

  • Hotel browsing with room details
  • Login & signup system
  • Booking confirmation page
  • Contact and reviews form

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB
